Drag And Drop™ Index

Drag and Drop™ (also called DnD™), is a visual scripting language that is based on the GameMaker Language (GML). It consists of using action blocks to construct game logic in a visual and intuitive way, and is an ideal tool for people who are learning programming, or for people that have a more visual inclination.
