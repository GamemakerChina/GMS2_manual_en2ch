Set Direction Fixed

This action is used to set a fixed movement direction. The available directions are indicated by the different buttons in the action block, and clicking on any of them will set that direction. Note that directions in GameMaker Studio 2 are measured in degrees, with right being 0°, up being 90°, left being 180° and down being 270°.
