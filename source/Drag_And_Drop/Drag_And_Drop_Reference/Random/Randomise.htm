Randomise

This action sets the "seed" for the random number generator to a random value. When using the random number functions in GameMaker Studio 2 the initial seed is always the same, as this makes tracing errors and debugging far easier (compiled games do not have this behaviour). Should you wish to test with true random, you should call this action once at the start of the game.
