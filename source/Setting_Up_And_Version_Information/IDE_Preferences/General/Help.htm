Help Preferences

The help file is designed to operate like a web page and even though the files are stored locally, it requires the use of an external browser to view them. The following settings related to the help file can be changed here:

  • Manual Language: This sets language for the IDE to use. The default is "English".
  • Help manual port: This sets the port number for the manual micro web server to use. The default value is 51291.
