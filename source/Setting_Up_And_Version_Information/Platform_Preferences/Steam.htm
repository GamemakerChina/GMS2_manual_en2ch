Steam Preferences

The Steam SDK is used for Windows, macOS and Ubuntu (Linux) builds that are to be distributed through the Steam platform. This section contains the setup options required to get it working within your projects:

  • Path To Steam SDK: The system path to where you have installed the Steam SDK. Once you have set this path to the SDK, you will then need to activate Steam in the Game Options and then for each platform add the App ID.
