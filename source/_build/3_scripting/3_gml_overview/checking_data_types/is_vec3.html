is_vec3


Description

In some cases you want to check and see what data type a variable holds in GameMaker Studio 2 and that's when you would use this function. It returns true or false depending on whether the value is a 3 value vector or not. For further information on data types see here.


Syntax:

is_vec3(n);


Arguments:

Argument Description
n The argument to check.


Returns:

Boolean


Example:

if !is_vec3(val)
   {
   show_debug_message("Not a 3 value vector!");
   }

The above code checks the variable "val" to see if it contains a vec3 and if it is not then it shows a message in the debug console.
