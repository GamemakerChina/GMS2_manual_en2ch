Extras

When working with GameMaker Studio 2 you will most often have to use the different Editors, but some of them have additional tools for using with them, and there are are also additional tools and windows within the IDE for specific tasks, like setting up different devices to compile to when testing your game projects. Below you can find a list of these additional tools:

  1. Sound Mixer
  2. The Output Dock
  3. Device Manager
  4. Remote Workers
  5. Project Image Generator
  6. The Debug Module
  7. Source Control
  8. The Marketplace
  9. Recent Windows
