The Tools Menu

This menu permits you to open the various stand alone tools that are available to you within GameMaker Studio 2, with the following options:

  • Texture Groups - View the Texture Group window.
  • Audio Groups - View the Audio Group window.
  • Sound Mixer - View the Sound Mixer window.
  • Generate Project Images - Opens the Generate Project Images window.
  • Create Local Package - Opens the Asset Packager window, so you can make and save local asset packages for future use.
  • Import Local Package - Opens a file explorer window where you can then select a Local Asset Package and import it, either adding it to the current project, or creating a new project for it.
