GameMaker Language Index

The GameMaker Language (also called simply GML) is the proprietary GameMaker Studio 2 scripting language. This language is structured to permit users to create their games in an intuitive and flexible way while offering all the power of any other major programming language. It is also the basis for Drag and Drop™ and can be used in conjunction with that if required.
