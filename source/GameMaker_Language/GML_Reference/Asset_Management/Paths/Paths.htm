Paths

GameMaker Studio 2 has an extensive collection of functions and built-in variables for using paths within your game. This section deals with the basic ones that are used when dealing with paths that have previously been created in the Path Editor, but further functions exist that permit you to create your own paths using code, and to manipulate them in various ways.

The following basic functions exist that deal with paths:

 

  • path_start
  • path_end
  • path_get_closed
  • path_get_kind
  • path_get_length
  • path_get_name
  • path_get_number
  • path_get_point_speed
  • path_get_point_x
  • path_get_point_y
  • path_get_precision
  • path_get_speed
  • path_get_x
  • path_get_y

 

To aid with debugging paths, you have a function to draw it to the screen too:

  • draw_path

 

Each instance of an object also has a series of built-in variables designed to make using paths easier, and these variables are listed in the following section:

  • Path Variables

 

Finally, it is possible to manipulate the contents of a path as well as create new paths when the game is running. The functions that enable these things and more are available from the following section:

  • Path Manipulation
