Sprite Instance Variables

These variables are included by default with GameMaker Studio 2 objects and are all related to the sprite assigned to the instance in the room. These variables can be accessed at any time within the code or actions for an instance and used to change what is drawn and how.

 

  • sprite_index
  • sprite_width
  • sprite_height
  • sprite_xoffset
  • sprite_yoffset
  • image_alpha
  • image_angle
  • image_blend
  • image_index
  • image_number
  • image_speed
  • image_xscale
  • image_yscale
  • mask_index
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top
