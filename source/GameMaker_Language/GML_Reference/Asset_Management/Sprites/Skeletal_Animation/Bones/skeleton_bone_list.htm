skeleton_bone_list

With this function you can populate a (pre-created) DS list with all the names of the bones used as part of the skeletal animation sprite. The names will be strings and can then be used in the other skeleton animation bone functions for these types of sprite.

 

Syntax:

skeleton_bone_list(sprite, list);

Argument Description
sprite The sprite index of the Spine skeletal animation to get the list from.
list The ID of the DS list to populate with the bone names.

 

Returns:

N/A

 

Example:

bone_list = ds_list_create();
skeleton_bone_list(sprite_index, bone_list);

The above code creates a DS list then populates it with the bone names for later use.
