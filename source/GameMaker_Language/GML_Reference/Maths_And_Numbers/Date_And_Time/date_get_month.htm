date_get_month

This function returns the month of the given datetime value.

 

Syntax:

date_get_month(date);

Argument Description
date The datetime to check.

 

Returns:

Real

 

Example:

mymonth = date_get_month(date_current_datetime());

This would set mymonth to the current month.
