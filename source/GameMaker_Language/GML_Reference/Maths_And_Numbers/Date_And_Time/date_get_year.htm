date_get_year

This function returns the year of the given datetime.

 

Syntax:

date_get_year(date);

Argument Description
date The datetime to check.

 

Returns:

Real

 

Example:

myyear = date_get_year(date_current_datetime());

This would set "myyear" to the current year.
