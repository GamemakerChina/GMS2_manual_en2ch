array_delete

With this function you can delete a value (or values) from an array at any given position. The function requires you to provide a variable that holds the array, the index (position) in the array to delete from, as well as the number of values to delete.

 

Syntax:

array_delete(variable, index, number);

Argument Description
variable The variable that holds the array.
index The index (position) in the array to delete the value(s) from.
number The number of values to delete.

 

Returns:

N/A

 

Example:

array_delete(score_array, 0, 10);

The above code will delete all the values in the given array from index 0 to index 9.
