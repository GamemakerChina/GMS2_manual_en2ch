is_numeric

This function returns whether a given variable is a numeric value (real, int32, int64 or boolean) or not. In some cases you want to check and see if a variable in GameMaker Studio 2 holds any numeric value, and that's when you would use this function. The function will return true if the given input is numeric, and false otherwise.

 

Syntax:

is_numeric(n);

Argument Description
n The input to check.

 

Returns:

Boolean

 

Example:

if is_numeric(val)
    {
    current_val += val;
    }

The above code checks the variable "val" to see if it is a numeric value and if it is it then adds it to the given variable.
