phy_speed_x

This variable can be used to get or to change the x component the instances linear speed vector and is defined in pixels per step (for pixels per second, see phy_linear_velocity_x). Altering this for a static instance (ie: an instance with 0 density) will turn it into a kinematic instance

 

Syntax:

phy_speed_x;

 

Returns:

Real (single precision floating point value, or undefined if the instance is not physics enabled)

 

Example:

if phy_speed_x != 0
    {
    phy_speed_x = 0;
    }

The above code will check the x component of the linear speed vector and if it is anything other than 0 it will set it to 0.
