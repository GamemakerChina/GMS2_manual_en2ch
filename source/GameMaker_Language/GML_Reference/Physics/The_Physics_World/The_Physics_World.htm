The Physics World

Before any physical reactions can occur in GameMaker Studio 2 you must first define the physics world in which they are going to happen. If you do not do this, GameMaker Studio 2 will default to the basic collision functions, and all other physics related code or settings will be ignored. You can set up the physics world from the Room Editor, but you can also set the same properties using code with the following functions:

 

  • physics_world_create
  • physics_world_gravity
  • physics_world_update_iterations
  • physics_world_update_speed
  • physics_world_draw_debug
  • physics_draw_debug
  • physics_pause_enable
