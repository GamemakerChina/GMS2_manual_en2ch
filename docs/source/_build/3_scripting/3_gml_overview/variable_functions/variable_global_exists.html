variable_global_exists


描述

使用此函数，你可以检查是否存在全局范围变量。 检查你提供 字符串 类型的全局变量名称（请参阅下面的示例代码），如果指定名称的全局变量存在，则函数将返回 true，否则返回 false。

注意： 该函数基本上是用于所导入工程的 兼容性脚本（Compatibility） 及拖拽按钮（Drag and Drop）动作转换而成的代码中的。


语法：

variable_global_exists(name);


参数：

参数 描述
name 待检测的全局变量名称（字符串形式）

返回：

Boolean（布尔值）


举例：

if !variable_global_exists("enemy_num")) global.enemy_num = instance_number(obj_Enemey_Parent);

上述代码将会检测名字叫 "enemy_num" 的全局变量是否存在，如果不存在则创建这样的全局变量并为其设定一个值。
