ptr


描述

此函数将尝试将给定值转换为指针数据类型，其值 必须 为实数、字符串、int64、int32 或 ptr。任何其他内容都会导致游戏崩溃并显示错误消息。


Syntax:

ptr(n);


参数：

参数 描述
n 要转换的值。

Returns:

指针


例如：

if !is_ptr(val)
   {
   ptr(application_surface);
   }

上述代码将会检测变量 “val” 来观察它是否存储一个指针，如果不是则将一个指针赋予给它。
