赋值


赋值语句把一个值赋予一个变量，其格式如下：

<变量> = <表达式>;

其中，表达式（expression）可以是一个值，也可以是复杂的代数式，除了将一个值赋给一个变量，还可以使用 “+=” 在变量存储的值上加一个值，比如：

a = 100;   // 指定一个简单的值
b = 200;
c = 300;
a += b;    // 使用操作符赋值
a = b + c; // 使用表达式赋值

类似地，你可以用 -= 实现自减，使用 *= 自乘，使用 /= 自除，或者也可以使用位运算符 |=、 &=和 ^=。你也可以用以下符号加减 一 ： ++，--。更多详细信息，请参阅相关 表达式 部分。

请注意你 不能 做以下操作（或任何变化的形式）：

var a, b, c;
a = b = c = 4;

而应该这样做：

a = 4;
b = 4;
c = 4;
