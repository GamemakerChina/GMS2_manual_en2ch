内置全局变量


GameMaker Studio 2 有一个 “内置” 全局变量的集合，其中大部分是非常具体的，只会在极少数情况下使用 - 这些在本手册的相应部分列出 - 但有一个重要的函数是经常使用的，没有在其他地方列出:

  • async_load

要注意，还有三个 弃用的 内置全局变量，你也应该知道（这三个变量仅用于支持从 GameMaker 前代版本中导入的传统项目，请不要在新项目中使用）：

  • score
  • health
  • lives
