播放音频


描述

This action will set the given sound playing. You select the sound from the Asset Explorer and then set whether you would like the sound to be looped or not (a looped sound will play again and again until stopped) by setting the "Loop" flag.


动作语法：


参数：

参数 描述
声音 The sound resource to play
循环 Whether to loop the sound or not (off by default)


举例：

The above action block code checks for a mouse button press and if one is detected it then checks a global variable to see if it is not true. If it is not true, then a sound is played (and looped) and the global variable set to true, otherwise the sound is stopped and the global variable is set to false.
