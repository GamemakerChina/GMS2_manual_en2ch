绘制实例的血条


描述

此动作可以绘制一个颜色条，用来展示实例的变量 健康值 (当你在 GameMaker Studio 2使用 任何 "健康值" 动作，使用这个动作的实例将会拥有一个新的实例t审阅变量称为 健康值 added)。被显示的数值应该为一个百分比值，数值为0-100. 你可以设置血条绘制的方向（从左到右，从右到左，从上到下，从下到上），还可以设置血条绘制的位置（在房间中的指定位置，或者在调用绘制动作的实例所在位置）你还可以设置4个角的混合颜色，创建出从红到绿的渐变色（只是举例，你可以使用任何颜色）下图展示了设置不同参数的血条的显示效果：

注意：这个动作只能用于各种 绘制动作，如果用在其它位置将不生效。


动作语法：


参数：

参数 描述
方向 绘制血条的方向
左 血条左端的位置
顶 血条顶端的位置
右 血条右端的位置
底 血条底端的位置
背景 血条背景颜色
轮廓 血条外轮廓的颜色
最小颜色值 最小值使用的颜色
最大颜色值 最大值使用的颜色
