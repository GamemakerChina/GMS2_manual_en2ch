Burst Particle Emitter


描述

This action will burst the given particles from the chosen emitter. You supply the emitter ID (as defined by the action Create Particle Emitter) as well as the particle ID to use (as defined by the action Create Particle Type) and a number of particles that you wish to burst. This action will create all the particles at once within the boundaries of the emitter shape and is useful for effects that need to be created once only (like explosions, or magic spells, for example).
