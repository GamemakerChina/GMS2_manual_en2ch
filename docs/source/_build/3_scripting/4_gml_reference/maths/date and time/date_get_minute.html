date_get_minute

返回与日期相对应的分钟数。

语法：

date_get_minute(date);


参数 描述
date 用于检查的日期时间。


返回： Real（实数）


描述

此函数返回给定日期时间的分钟数。


例如：

myminute = date_get_minute(date_current_datetime());

这将把 "myminute" 设为当前的分钟数。
