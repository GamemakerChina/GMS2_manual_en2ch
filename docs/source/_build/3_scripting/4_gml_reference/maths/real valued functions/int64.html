int64


描述

该函数将尝试将给定的值转换为64位整数，其中的值 必须为实数、字符串、int64、int32或ptr。任何其他参数都会导致游戏崩溃并显示错误消息。您可以使用函数 is_int64检查变量是否为64位整数。


语法：

int64(val);


参数 描述
val 要转换的值。


Returns:

64位整数


例如：

steam_handle = int64(global.fileReadString);

上面的代码将全局变量中的值转换为64位整数。
