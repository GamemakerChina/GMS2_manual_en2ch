exp


描述

此函数和函数power（e，n）的功能是一样的，其中e约为2.718281828（也称为欧拉数），n是它应该乘以它的次数。总的来说，就是“在n个单位时间后（连续增长）我会增长多少？”。所以 exp(3) 会返回 20.08，这意味着在3个时间单位之后，会增长到一开始的20.08倍。


语法：

exp(n);


参数 描述
n e的次方。


Returns:

实数


例如：

val = exp(2);

这将val设置为e ^ 2，大约为7.4。
