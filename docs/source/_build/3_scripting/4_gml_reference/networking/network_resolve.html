network_resolve

Syntax:

network_resolve(url);


参数 描述
url 待获取IP的URL（字符串形式）。


返回： String（字符串）


描述

该函数将会返回指定URL的IP地址。


例如：

game_ip = network_resolve("www.macsweeneygames.com");

上述代码将会返回所指定的URL的IP地址并将其存入变量“game_ip”中。
