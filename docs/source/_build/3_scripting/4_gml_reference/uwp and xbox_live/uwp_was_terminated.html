uwp_was_terminated


描述

With this function you can find out if the game was previously shut down by the system (i.e. itâ€™s not the first run of the game). If it has been then the function will return true, otherwise it will return false. An example of use would be to check it and see if the game was previously terminated by the system, and then (if it returns true) to load the previous stage that the player was at in the game. If targeting the XBox One then the required behaviour of games if this is detected will be different between the different versions of the certification docs, so itâ€™s worth checking the latest version to see what the game should do.


Syntax:

uwp_was_terminated();


Returns:

Boolean


例如：

if xboxlivewas_terminated()
    {
    buffer_load("Player_Save.sav");
    global.GameLoaded = true;
    }

The above code checks to see if the app has previously been terminated by the system, and if so, it loads a buffer and sets a global variable.
