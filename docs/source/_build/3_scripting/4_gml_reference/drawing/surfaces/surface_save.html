surface_save


Syntax:

surface_save(surface_id, fname);

参数 描述
surface_id 作为绘制目标的表面层的ID。
fname 保存的图像文件的名称。


Returns:

N/A（无返回值）


描述

此函数将使用给定的文件名保存表面层到硬盘上。表面层 只能 保存为 *.png 格式文件。


例如：

if keyboard_check_pressed(ord("S")
   {
   surface_save(surf, "test.png");
   }

上面的代码将检查是否用户按下键盘上的“S”键，如果他们这样操作了，将对变量“surf”索引的表面层存盘保存。


返回: 表面层
