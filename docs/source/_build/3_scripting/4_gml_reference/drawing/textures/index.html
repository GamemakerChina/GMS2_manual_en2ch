There are also a few special functions relating to the way that GameMaker Studio 2 draws the textures pages that it uses. Please realise that a sprite is actually a textured quad (a four point primitive made of two triangles) and as such some of the following functions will also affect how they are drawn:

  • texture_get_height
  • texture_get_width
  • texture_get_texel_height
  • texture_get_texel_width
  • texture_get_uvs
  • texture_set_stage
  • draw_texture_flush

There are also a few sprite functions related to how you deal with their textures too:

  • sprite_flush
  • sprite_flush_multi
  • sprite_prefetch
  • sprite_prefetch_multi
