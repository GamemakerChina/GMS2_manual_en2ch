timeline_clear

清除时间轴的所有时刻

语法：

timeline_clear(ind);


参数 描述
ind 要清除时刻的时间轴索引。


返回： N/A（无返回值）


描述

使用此函数，你可以清除特定时间轴的 “时刻”，删除该时间轴的所有代码和操作，并将其清空。


例如：

if timeline_position > 200
   {
   timeline_clear(global.tl);
   timeline_index = -1;
   }

上面的代码将在一个特定的时刻已经过去时清除由变量 “global” 索引的指定时间轴。
