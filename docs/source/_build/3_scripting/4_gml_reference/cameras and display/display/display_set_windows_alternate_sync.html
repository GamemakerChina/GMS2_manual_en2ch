display_set_windows_alternate_sync

设置垂直同步的方法（只适用Windows）。

语法：

display_set_windows_alternate_sync(flag);


参数 描述
标识 使用替代的方法（true）或者常规的方法（false）。


返回：N/A


描述

这个只用于 Windows 的函数可以用于在运行时改变垂直同步方法。当设置为 true （并且激活了垂直同步）时，替代方法将被使用，取代默认的方法，以减少任何可能的视觉延迟。注意，你也可以在全局游戏设定中设置并且只适用 Windows 7 或更高的系统 - XP 或者 Vista 上，勾上这个选项也将使用默认的方法 - 并且它 不会 开启或关闭垂直同步，因为那个你将需要使用到函数 display_reset()。


举例：

if !display_get_windows_alternate_sync()
   {
   display_set_windows_alternate_sync(true);
   }

上述代码检测是否使用于替代的垂直同步方法，如果是默认的则切换为替代的方法。
