window_set_fullscreen

设置游戏是否全屏运行或者窗口运行。

语法：

window_set_fullscreen(full);


参数 描述
全屏 设置屏幕为全屏（true）或不是（false）。


返回： N/A（无返回值）


描述

有了这个函数，你可以设置游戏窗口为全屏幕（true）或者不是全屏（false）。请注意对于 Mac OSX 目标平台， 你 必须选中 "Start In Fullscreen" 选项，该选项在 全局游戏设置中可以找到，否则这个函数将使用无效。. 还请注意除非作为一个“可单击”回调添加，否则此函数 不会 在HTML5中起作用（参阅 这儿来获得更多信息）。


例如：

if mouse_check_button_pressed(mb_left)
   {
   if window_get_fullscreen()
      {
      window_set_fullscreen(false);
      }
   else
      {
      window_set_fullscreen(true);
      }
   }

上述代码将检测鼠标按下，如果还不是全屏则设置窗口为全屏，否则在已经是全屏模式时设置为窗口。
