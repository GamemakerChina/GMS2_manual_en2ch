window_set_position

设置浏览器内或者屏幕的可见区域内的窗口位置。

语法：

window_set_position(x, y);


参数 描述
x 窗口位置的x坐标。
y 窗口位置的x坐标。.


返回：N/A


描述

有了这个函数，你可以设置游戏窗口至显示器内的特定位置 （适用 MacOS, Linux（Ubuntu） 和 Windows） 或者浏览器内 （HTML5）。

注意: 如果你的 HTML5 游戏使用用户自定义的index.html ，并且设置画布（canvas）至固定的位置，那么这个函数在窗口位置将无效。


举例：

window_set_position(0, 0);

上述代码将放置游戏窗口至浏览器或者显示器（根据正在使用的目标模块）的左上角。
