window_set_size

设置窗口的大小，以像素为单位。

语法：

window_set_size( w, h );


参数 描述
w 窗口的新宽度。
h 窗口的新高度。


返回：N/A


描述

有了这个函数，你可能更改游戏窗口的大小。如果你使用这个函数改变游戏窗口口的大小，窗口内容将与背景画布以1：1的比例进行调整，以保持图像的清晰和明确（尽管缩放窗口不同于房间或视野）。比如，你的窗口是 200 x 150 像素，玩家拖动边界拉伸窗口，最后窗口可能会是400 x 300 像素，但是绘制画布仍然是 200 x 150，因为像素缩放比例为1:2，图像变得模糊。尽管如此，如果你使用这个函数设置窗口大小，即使图像自适应至新的窗口大小，图像与窗口的缩放比例仍然保持1:1。


例如：

if mouse_check_button_pressed(mb_left)
   {
   global.scale += 1;
   window_set_size(200 * global.scale, 150 * global.scale);
   }

上述代码将在鼠标按下时，基于变量“global.scale”改变窗口的大小。
