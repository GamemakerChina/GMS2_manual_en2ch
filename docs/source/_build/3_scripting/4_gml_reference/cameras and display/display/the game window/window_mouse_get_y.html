window_mouse_get_y

返回窗口内的鼠标y坐标。

语法：

window_mouse_get_y();


返回： Real（实数）


描述

使用该函数你可以获取到鼠标光标在显示器上（Windows、Ubuntu(Linux)及MacOS平台）的y坐标（像素为单位），如果是HTML5平台则是在浏览器中的位置。

注意：对于常规的鼠标函数请参见鼠标输入部分。


举例：

wy = window_mouse_get_y();

上述代码使用变量"wy"记录了鼠标在窗口中的y轴位置。
