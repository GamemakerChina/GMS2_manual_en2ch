window_handle

返回游戏窗口的id。

Syntax:

window_handle();


返回： 指针


描述

使用该函数你能获取到内部Windows内部id值（HWND，一个指针）。This function is really only useful for extension writers who need the window handle to call Windows API's in DLL code (the returned pointer should be cast into a string and then in the C++ just cast it to an HWND). The table below shows the platforms supported along with what they return:

平台 返回
Windows Window HWND
macOS NSWindow类
Ubuntu (Linux) XWindow句柄
HTML5 Canvas ID


例如：

win_id = windows_handle();

上述代码将游戏窗口id存入一个变量。
