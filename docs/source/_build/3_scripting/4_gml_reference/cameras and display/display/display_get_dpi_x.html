display_get_dpi_x


描述

每英寸打印的点数（DPI）是空间印刷或者视频点密度的量度，特别是指放置在一英寸（2.54 cm）直线跨度内的独立点的数目。在移动设备上运行时（尤其是安卓设备），这是一个重要的考量，适合这个显示器的分辨率时，可能不适合另外一个。比如，你可能有两个显示器，它们有相同的分辨率400 x 800，但是显示器1是60dpi，而显示器2是30dpi。在这种情况下，即使实际分辨率相同，显示器2上显示的任何文本或图像也会显得更大。

这个函数将获取设置显示器x轴的dpi（这个值也依赖设备的方向）。请注意，Mac和iOS不会返回特定的dpi设置，虽然看起来返回与操作系统相同的值，但这些值不正确（但必须这样做），因为Apple没有提供正确的值。


语法：

display_get_dpi_x();


Returns:

实数


例如：

dpx = display_get_dpi_x();

设置变量“dpx”等于x轴的dpi值。
