view_set_visible


描述

上述代码将设置给定可视区域的可见性。此函数需要一个可视区域索引（从0到7）和一个布尔值 ：如果可见则为true ， 不可见则为false。


语法：

view_set_visible(view_port)


参数：

参数 描述
view_port The view port to target (0 - 7)
visible Visibility toggle (true is visible and false is invisible)


Returns:

N/A（无返回值）


例如：

if !view_get_visible(0)    {
   view_set_visible(0, true);
   }

上述代码检查view port[0]是否可见，如果不可见，则将它设为可见。
