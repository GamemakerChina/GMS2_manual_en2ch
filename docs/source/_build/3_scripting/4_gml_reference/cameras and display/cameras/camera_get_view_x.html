camera_get_view_x


描述

This function can be used to retrieve the x position of the view for a given camera.


语法：

camera_get_view_x(camera_id)


参数：

参数 描述
camera_id The unique camera ID value returned when you created the camera


返回：

实数


举例：

var vx = camera_get_view_x(view_camera[0]);
var vy = camera_get_view_y(view_camera[0]);
draw_text(vx + 5, vy + 5, "SCORE" + string(score));

The above code retrieves the position of the camera assigned to view port[0] and then draws text relative to that position.
