Controls


制作一个 游戏最根本的事件是与玩家交互。有多种方法可以实现交互，比如电脑的键盘、鼠标，移动设备的触摸和倾斜。在 GameMaker 中通常的做法是在鼠标或按键事件中放置动作，但有时你需要更多的控制，尤其是移动设备或 HTML5。For that you can use the mouse and keyboard functions to get which buttons have been pressed or released as well as retrieve the touches on a screen using the gesture functions, and there are a couple of specific functions relating to the tilt controls that most mobile devices have too.

The following sections deal with the main user input devices for developing games with GameMaker Studio 2:

  1. 键盘输入
  2. 鼠标输入
  3. 设备输入
  4. 手势输入
  5. GamePad Input
  6. 虚拟按键
