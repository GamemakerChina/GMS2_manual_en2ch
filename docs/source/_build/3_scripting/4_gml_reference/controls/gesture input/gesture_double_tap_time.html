gesture_double_tap_time


描述

此函数用于设置两次触摸/点击之间触发 双击手势 事件所需的时间。时间以秒为单位，默认值为 0.16。


语法：

gesture_double_tap_time(time);


参数 描述
time 设置双击手势事件检测的时间（以秒为单位）。


Returns:

N/A（无返回值）


例如：

if gesture_get_double_tap_time() != 0.16
    {
    gesture_get_double_tap_time(0.16);
    }

上面的代码检查手势的双击时间是否设置为 0.16 秒，如果不是，则将其设置为该值。
