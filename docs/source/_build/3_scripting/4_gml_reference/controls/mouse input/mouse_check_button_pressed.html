mouse_check_button_pressed


描述

This function will return true if the mouse button being checked has been pressed or false if it has not. This function will only be triggered once for any mouse button when it is first pressed and to trigger it again the button will need to have been released and pressed again. Note that it will be considered triggered for the duration of the step, and for all instances that have any mouse events or that use this same function.

You supply the mouse button to check from one of the following constants:

常量 描述
mb_left The left mouse button
mb_middle The middle mouse button (this may not be valid for all target platforms)
mb_right The right mouse button
mb_none No mouse button
mb_any Any of the mouse buttons


语法：

mouse_check_button_pressed(numb);


参数 描述
numb Which mouse button constant to check for.


Returns:

Boolean


例如：

if mouse_check_button_pressed(mb_left)
   {
   score += 50;
   }

The above code will check to see if the left mouse button has been pressed and if it has it will add 50 to the score.
