string_copy


描述

使用此函数，你可以轻松地从字符串中选择要复制到另一个字符的字符数。字符串中的第一个字符始终编号为 “1”，因此要复制（例如）字符串的前五个字符，你应该编写 string_copy(str, 1, 5)。下面提供另一个例子。


语法：

string_copy(str, index, count);

参数 描述
str 要复制的字符串。
index 要复制的字符串中第一个字符的位置（从 1 开始编号）
count 从第一个位置开始要复制的字符数


返回：

String（字符串）


例如：

name = keyboard_string;
if string_length(name) > 15 )
      {
      keyboard_string = string_copy(name, 1, 15 );
      }

上面的代码将允许玩家通过键盘输入一个字符串，然后将其存储在变量 name 中。然后检查此变量以查看它是否超过 15 个字符的最大长度，如果超过，则使用 name 字符串的 15 个字符副本替换 keyboard_string。这有效地将玩家输入限制为 15 个字符。
