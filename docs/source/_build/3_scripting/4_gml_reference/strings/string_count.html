string_count


描述

此函数将返回给定子字符串在特定字符串中出现的次数。通过这种方式，你可以检查单个字母或短语在存储文本部分中重复的次数。


语法：

string_count(substr, str);

参数 描述
substr 用于检查字符串的子字符串。
str 要检查的字符串。


返回：

Real（实数）


例如：

str1 = "Hello World";
ocount = string_count( "o", str1 );

这会将变量 ocount 设置为 str1 中的 “o” 出现的次数，在本例中为 2。
