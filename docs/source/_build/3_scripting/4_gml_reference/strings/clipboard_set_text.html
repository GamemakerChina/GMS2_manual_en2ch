clipboard_set_text


描述

此函数将设置剪贴板以保存指定的字符串。你可以将其设置为空字符串 “” 以有效清除剪贴板的文本。

注意： 此函数仅对 Windows 目标有效。


语法：

clipboard_set_text(string);

参数 描述
string 用于设置要写入剪贴板的文本（作为字符串）。


返回：

N/A（无返回值）


例如：

if clipboard_has_text()
   {
   str = clipboard_get_text();
   clipboard_set_text("");
   }

上面的代码检查剪贴板中的文本，如果它包含任意字符串，则将其作为字符串读入变量 “str”。最后，通过将剪贴板设置为空字符串来清除剪贴板。
