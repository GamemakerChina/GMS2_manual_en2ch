string_ord_at


描述

You can use this function to return a specific character code at a specific position within a string, with the index starting at 1 for the first character. If no character is found or the string is shorter than the value given to index, an empty string "" is returned.


Syntax:

string_ord_at(str, index);

参数 描述
str The string to check.
index The position to get the character code from.


Returns:

实数


例如：

str = "Hello World";
char_code = string_ord_at(str1, 7);

This will get the character code for the seventh character (where "H" counts as the first) in string "str" and store it in the variable "char_code".
