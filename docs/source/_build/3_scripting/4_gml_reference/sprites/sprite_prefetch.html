sprite_prefetch


Syntax:

sprite_prefetch(ind)

参数 描述
ind 将要被预读取的精灵的所有。


Returns:

N/A（无返回值）


描述

此函数可以用来预读取给定精灵的纹理页，并将它放进纹理存储中。你要提供在创建精灵资源时定义的精灵索引，然后此精灵所在的纹理页将会被加载进存储中。.

注意： 在多数设备上，将纹理放进纹理存储会暂时降低性能。所以推荐你使用 sprite_preftech 在游戏关卡开始就预读取需要用到的图像来避免性能降低。


例如：

sprite_prefetch(spr_Player_Aura);

上述代码将会把给定的精灵放进纹理存储中并供将来使用。
