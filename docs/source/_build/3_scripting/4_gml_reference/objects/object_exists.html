object_exists

返回具有给定索引的物体是否存在。

语法：

object_exists(obj);


参数 描述
obj 待检测的物体索引。


返回： 布尔值


描述

该函数返回指定索引的物体是否存在。请注意这个函数检查的是该物体是否出现在资源树中而非实际的游戏房间中。为此，你应该使用函数 instance_exists。


例如：

if object_exists(obj_temp)
   {
   with (obj_Temp) scr_add_event();
   }

上面的代码检查变量 “obj_temp” 以确保它存储物体的索引，然后，如果存在，则使物体运行脚本。
