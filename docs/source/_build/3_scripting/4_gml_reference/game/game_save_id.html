game_save_id

语法：

game_save_id


返回: String（字符串）。


描述

此 只读 变量将返回游戏用于保存文件的目录的完整路径 ID。此目录可能对其他应用程序可见，也可能不可见，具体取决于平台，并且在 HTML5 目标上，它将返回空字符串。

注意：此变量仅供参考，不 用于获取文件路径或其他类似内容，因为它无法在所有平台上正常工作。




例如：

save_dir = game_save_id;

这将存储用于将文件保存到变量的目录以供将来使用。
