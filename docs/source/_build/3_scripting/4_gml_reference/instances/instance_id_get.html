instance_id_get


描述

通过这个函数，你可以从当前活动的实例列表中获得任何实例的惟一 ID 值。在实例列表中给出索引以获取 ID，函数将返回存储在变量中的值。


语法：

instance_id_get(index);


参数：

参数 描述
index 实例列表中从0到（实例数 - 1）的索引。


返回：

Instance ID value (real)（实例 ID 值，实数）


例如：

for (var i = 0; i < instance_count; i ++;)
    {
    var temp_id = instance_id_get(i);     with (temp.id) speed += 0.1;
    }

上面的代码将遍历房间内的所有实例，并增加 0.1 的速度。
