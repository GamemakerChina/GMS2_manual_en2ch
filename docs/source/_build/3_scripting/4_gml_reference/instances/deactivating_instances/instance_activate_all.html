instance_activate_all


语法：

instance_activate_all();


返回： N/A（无返回值）


描述

With this function you can tell GameMaker Studio 2 to activate all instances that have been previously deactivated in a room.


举例：

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, _vx - 64, _vy + 128, view_hview + 128, false, false);

The above code activates all instances within the room and then deactivates those that are outside of the limits of the current camera view.
