实例


实例（Instances） 是从 物体（Objects） 创建的，一旦创建，就可以通过 GML 代码和 DnD™ 动作进行更改和操作。下面几节将介绍所有实例都具有的内部 内置变量，以及专门用于查找和操作实例的函数，以及 GameMaker Studio 2 中用于限制房间中“活动”实例数量的例程。

你可以从以下部分中找到更多信息：

  1. 实例变量
  2. 实例函数
  3. 冻结实例

注意，有两个额外的内置变量（它们是 全局作用域（global scope） 变量）和一个与使用实例相关的函数，可以用来检索某些细节：

  1. instance_id
  2. instance_count
  3. instance_id_get
