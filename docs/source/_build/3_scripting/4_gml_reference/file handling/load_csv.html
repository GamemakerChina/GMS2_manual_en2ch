load_csv


描述

此函数将加载 CSV 格式文件并将其转换为栅格数据，返回创建的栅格的唯一 ID 值。


语法：

load_csv(filename)


参数 描述
filename 要打开的文件的名称（字符串）


Returns:

DS Grid Index (real)


例如：

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_width(file_grid);
var hh = ds_grid_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++;)
    {
    for (var j = 0; j < hh; j++;)
        {
        draw_text(xx, yy, string(file_grid[# i, j]));
        yy += 32;
        }
    yy = 32;
    xx += 32;
    }

上面的代码将打开给定的 CSV 文件，并将返回的栅格数据存储在变量 “file_grid” 中。 然后，这个栅格被两个 for 循环解析并且将内容绘制到屏幕上。
