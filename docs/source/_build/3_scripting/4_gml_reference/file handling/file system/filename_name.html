filename_name

语法：

filename_name(fname);


参数 描述
fname 要使用的文件。


返回: String（字符串）。


描述

使用此函数返回指定文件的名称部分，带 扩展名，但 不带路径


举例：

name = filename_name(file_find_first("C:\Games\*.doc", 0));

上面的代码获取在指定目录中找到的第一个 “doc” 类型文件的名称（作为字符串）。
