ini_write_string

语法：

ini_write_string(section, key, value);


参数 描述
section 写入 .ini 的键所在的节。
key .ini 要写入的键。
值 要写入指定键的字符串。


返回： N/A（无返回值）


描述

你可以使用此函数将字符串（文本）写入 ini 数据文件。Ini 文件由 节（sections） 组成，然后每个节由 键 - 值 对（key - value pairs）组成。所以典型的 ini 文件看起来像这样：


举例：

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

上面的代码打开一个用于读写的 ini 文件，然后在关闭文件之前将存储在全局变量 “Name” 中的字符串写入 “Save” 节的 “Player” 键。
