ini_read_string

语法：

ini_read_string(section, key, default);


参数 描述
section 要读取值的键所在的 .ini 节。
key 要读取值的键。
default 如果在定义的位置找不到字符串（或 .ini 文件不存在），则返回的默认字符串。必须是一个字符串。


返回： String（字符串）


描述

你可以使用此函数从 ini 数据文件中读取字符串（文本）。Ini 文件由 节（sections） 组成，然后每个节由 键 - 值 对（key - value pairs）组成。所以典型的 ini 文件看起来像这样：


举例：

ini_open( 'savedata.ini' );
global.name = ini_read_string( 'save1', 'Name', 'Player1' );
ini_close();

这将打开 'savedata.ini' 并将 global.name 设置为其中 'save1' > 'Name' 键的字符串，然后再次关闭 .ini。如果 'save1' > 'Name' 下没有字符串，或者没有 'savedata.ini' 文件，global.name 将被设置为 'Player1'。
