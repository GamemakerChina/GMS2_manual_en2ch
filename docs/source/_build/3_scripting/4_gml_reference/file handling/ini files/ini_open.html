ini_open

Syntax:

ini_open(name);


参数 描述
name ini 文件的文件名


返回： N/A（无返回值）


描述

这将打开一个用于读取和 / 或写入的 ini_file。如果你正在检查的位置不存在 ini_file，GameMaker Studio 2可能会创建一个，但前提是你向其写入数据。如果你只读取了 ini 文件中的信息，那么将返回读取函数的默认值，但实际上 并没有 实际创建 ini 文件。

请注意，你一次只能打开 一个 ini 文件，并且在完成读取 / 写入后，请记得使用 ini_close .ini 文件，因为信息在此之前实际上并未存储到磁盘（它也存储在内存中，直到文件关闭）。


警告！ 此函数可能无法正常工作！ 因为 GameMaker Studio 2 被沙盒化。有关更多信息，请参阅 文件系统限制 一节。


例如：

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

这将打开 'savedata.ini' 并读取 “save1” 节中的 “score” 键代表的分数值，然后再次关闭 .ini。如果 “save1” 节 “score” 键没有值，或者没有 “savedata.ini” 文件，则得分将设置为 0（默认值）。
