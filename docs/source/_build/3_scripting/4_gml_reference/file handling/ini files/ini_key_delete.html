ini_key_delete

Syntax:

ini_key_delete(section, key);


参数 描述
section 要删除的键所在的节。
key 要删除的键。


返回： N/A（无返回值）


描述

使用此功能，你可以从 ini 文件中删除所选的键（及其对应的值）。


例如：

ini_open('savedata.ini');
ini_write_real('save1','Score',score);
ini_key_delete('save1','Score');
ini_close();

此示例将打开 'savedata.ini' 并将值写入 'save1' > 'Score'。然后它将删除 “Score” 键并关闭 .ini 文件。
