md5_file

Syntax:

md5_file(filename)


参数 描述
filename 用于生成 MD5 散列的文件。


返回： String（字符串）


描述

在密码学中，MD5（Message-Digest algorithm 5，消息摘要算法 5）是广泛使用的具有 128 位散列值的加密散列函数，并且已经在各种安全应用程序中使用。 它也常用于检查文件和字符串的完整性。 此函数将获取给定文件并为该文件生成唯一的 MD5，然后可以将其存储以供以后使用。


例如：

hash = md5_file(working_directory + "\game_data.ini")

上面的代码将为指定的文件生成 MD5 散列，并将返回的值存储在变量 “hash” 中。
