帮助菜单（Help）


该菜单是有关在使用GameMaker Studio 2期间获得帮助的内容。包含如下菜单项：

  • 起始页 - 该选项会打开一个新的起始页工作区

  • 欢迎页 - 该选项将会在一个新的工作区内打开一个初始欢迎滑动页面。

  • 打开帮助文档 - 打开帮助文档。注意，在菜单条中点击帮助按钮或按下键也可以打开帮助文档。

  • 知识库 - 这将带你进入 GameMaker Studio 2 帮助中心，在这里你可以找到有关平台设置，解决已知问题，如何使用 GameMaker Studio 2 及其它一些事。

  • 报告 BUG - 如果你在使用 GameMaker Studio 2 时发现任何错误或感觉存在任何问题，你可以 在此处提交。

  • YoYoGames 官网 - 打开 YoYo Games 主页。

  • GameMaker 社区 - 在浏览器中打开 GameMaker 交流论坛。请注意，这是一个社区论坛，所以如果你有错误要报告或需要技术帮助，你应该到官方支持论坛（见上文）。

  • 版本说明 - 该选项会在你的浏览器中打开 GameMaker Studio 2 的 版本发行信息。如果你想回滚一次更新，你可以通过点击页面中对应版本号的链接来获取之前版本的安装包。

  • 当前运行库版本说明 - 由于运行库可独立于IDE进行更新，因此它有自己的一套 运行库发行说明。选择此选项将在你的浏览器中打开说明，但请注意，当前说明中提到的运行库可能不是你正在使用的版本。要了解你所使用的运行库，或有关如何添加，删除以及激活运行库的信息，请参阅 运行库首选项 部分。

  • 必需的 SDK - 由于GameMaker Studio 2跨平台开发的本性，有一些第三方的开发工具包（SDK）是必需的。点击该选项将会带你到一个 服务支持论坛的文章，里面详细阐述了对于每个平台基底你所需要的东西。

  • 在资源管理器中打开项目 - 该选项将会在操作系统的文件资源管理器中打开保存当前被载入项目的位置。

  • 关于 - 打开 “关于 GameMaker Studio 2” 窗口，你可以在这里面获取到产品的版本信息（在上报 Bug 和问题时这很有用）。
