微型网络服务器


微型网络服务器（Micro Web Server,MWS）是一个主要用于连接 GameMaker Studio 2 可用的目标平台（如 HTML5 平台）及向目标平台添加伺服文件的小工具。当你在需要使用微型网络服务器的目标上运行游戏的时候，你会看到当游戏在所选设备上被伺服时会打开一个新的窗口：

注意：当你首次通过 MWS 在目标平台上运行游戏时可能会得到防火墙警报。如果发生这种情况，你应该允许网络服务器穿过防火墙进行通信。


这个网络服务器会显示一定量的与所伺服文件相关的调试信息，以及底部的一个用来清空这些信息的按钮。你不可以编辑此处的大部分信息（因为它们是通过目标管理器设备（Target Manager Devices）被设定好的，但是你可以修改网络速度（Network Speed）值）。Setting it to anything other than the maximum value will effectively throttle the nework transfer speed and permit you to test games in a low bandwidth environment.
