版本说明


每个版本的GameMaker Studio 2 都会有它自己的一套发行说明。这些说明包含了每个版本所改动的细节以及修复的Bug的完整列表。这是一个非常有用的工具，用于决定是否更新当前版本，否则在更新后你会发现一些东西可能是bug（译者注：做大工程时更新要谨慎，一定要仔细阅读版本说明，否则会有无法意料的后果。不过现在是自动更新，steam版可以在属性-测试中选择之前的版本）。
