偏好预设 - 时间轴


时间轴偏好设置 用于设置使用 时间轴编辑器 时的某些属性。可用的选项有：

  • 自动响应 DnD™ 转换为 GameMaker Language：当你在时间轴上使用鼠标右键 菜单选项将拖放模块转换为脚本（代码）时，你会收到一个提示，询问你是否要继续，因为这是一个单向转换。将此选项更改为“是”会取消显示该消息，并继续进行转换，就像在对话框中单击“是”一样。默认设置为“显示消息”。

  • 自动响应永久（permanent）动作警告：时间轴中的某些操作无法撤消，如果发生这种情况，你将收到一条警告消息，询问你是否要继续。将此选项更改为“是”将会取消显示该消息，并继续执行该操作，就像你在对话框中单击“是”一样。默认设置为“显示消息”。

  • 创建时立即打开新的时刻点开始编辑：通常，当你向时间轴添加新的时刻时，GameMaker Studio 2 将自动打开编辑器，以添加代码或拖放模块到其中。但是，如果你关闭此选项，那么添加新时刻时将不会打开编辑器（你可以通过双击 打开它）。默认情况下，此选项打开。
