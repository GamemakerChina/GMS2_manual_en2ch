偏好设置 - 图像编辑器（Image Editor）


图像编辑器偏好设置（Image Editor Preferences）被用于定义精灵和图像编辑器（Image Editor）窗口的外观和感受。下述为常规选项：

  • 默认循环模式（Default Loop Mode）：图像编辑器中的循环按钮可以设置#?两种#?功能模式：一种是单次播放（Play Once），这种情况下动画会在到达最后一帧时停止；另一个是从起点开始循环（Loop from the start），这种情况下动画会在结束后回到起点并继续播放；还有一个是往返播放（Ping pong），这种情况下动画会从头播放到尾再倒过来播放一遍回到开始。此处默认值为“循环”（Loop）

  • 绘制细的选区边框（Draw thin selection outline）：如果你勾选此项，则选区的边框将会是无视图像缩放的绝对的1像素宽，而解除此选项则使得边框为相对于图像缩放的1像素宽。该选项默认勾选。

  • Selection outline primary colour: The section outline is a dashed line that alternates between two colours. 该选项允许你选择这两种所用颜色的第一种颜色（包括透明度），默认值为#FFFFFD0。

  • Selection outline secondary colour: The section outline is a dashed line that alternates between two colours. 该选项允许你选择这两种所用颜色的第二种颜色（包括透明度），默认值为#0000D0。
  • Editor background colour: While editing a sprite frame the image may be scaled in or out or may not fit the whole drawing in window, in which case a background colour will be shown which you can set here. 默认值为#282828FF.

图像编辑器偏好设置中还有如下的子目录：

此处你可以为你编辑精灵图片下面的背景做以设定。默认地，网格为灰色正方形，但你可以用下列设定来改变它：

  • Canvas style - Here you set the "style" for the background, which can either be "Block Pattern" or "Single Colour". 单色模式会单纯地把背景设置为你所选的颜色（使用颜色 1（Colour 1）设定，下文会提到），而默认选项是方块图案（它还可以通过下文所示的其余对话框选项进行修改）。默认选项是“方块图案”

  • Canvas block size - When you have selected Block Pattern for the background, you can set the size of the individual block here. 默认为16像素，并允许最小值为2像素，最大值为256像素。

  • Canvas primary colour - If you have Single Colour chosen for the background, then the colour used is this one. 你可以点击颜色卡来打开颜色编辑器去改变它。当使用方块图案选项时，该颜色被设置为交替方块的第一种颜色。默认值为(RGBA颜色) #c3c3c3ff。

  • Canvas secondary colour (block pattern style) - If you have Block Pattern chosen for the background, then the colour used to alternate with colour 1 (above) is this one. 你可以点击颜色卡来打开颜色编辑器去改变它。默认值为(RGBA颜色) #808080ff。

在这里你可以设置当你对当前精灵执行各种动作时的确认行为。选项有：

  • Automatic response to deleting frames - When you delete a frame by default a message shows to ask if you are sure you want to do this. 设置此项为“是”（OK），你就可以去掉这些消息并且在接下来没有任何消息显示的情况下就可以删除帧。默认选项为“显示消息”（Show message）。

  • Automatic response to replacing images during strip import - When you select Import Strip from the Image drop down menu, you will get a message warning you that you will be overwriting any images already in the sprite resource, with the option to continue or not. 你可以在此处设置是否显示这个消息，还是让IDE直接选择“是”（Yes）或“否”（No）的选项。默认值为“显示消息”。

  • Automatic response to committing changes on tool change - When you change draw tools after editing a frame, by default a message shows to ask if you want to commit the changes made. 设置此项为“是”（OK），你就可以去掉这些消息并且在接下来当你改变工具的时候没有任何消息显示的情况下就提交修改。如果你选“否”（No），则帧编辑器会在你改变工具时静默地取消修改。默认选项为“显示消息”（Show message）。

  • Automatic response to locked layer warning - If you try to edit a locked layer you'll get a message by default informing you that you can't do this. 如果设置此项为“是”（OK），则不会显示该消息，并且无法做这样的编辑。

  • Automatic response to group selected warning - If you select multiple layers in the layer editor and then try to draw or edit then you will be shown a message by default to inform you that it is not possible. 你可以在此处选择“是”来去掉这个消息并无法一起编辑多个图像，此时只有在选择单个图层的时候才可以正常编辑。默认选项为“显示消息”（Show message）。


网格偏好设置（Grid Settings preferences ）被用于设置图像网格会如何显示，有如下可用选项：

  • Grid horizontal spacing (px) - This value sets the spacing for the horizontal lines of the image grid. 默认值为1像素。

  • Grid vertical spacing (px) - This value sets the spacing for the vertical lines of the image grid. 默认值为1像素。

  • 显示网格（Display Grid） - 勾选此项则图像编辑器会自动带上网格地打开图像。默认为关。

  • 吸附到网格（Snap To Grid） - 如果勾选此项，图像编辑器就会自动地将某些工具吸附到网格上。默认为关。

  • Grid line Colour - Here you can set the colour (including alpha value) for the image grid. 默认值为#000000FF.

在这里你可以设置图像编辑器中一些工具的操作和显示方式的各种选项。

如果你使用了弧线工具（Arc Tool）那么你可以在这里设置用于绘制节点属性的各种颜色：

  • Anchor point colour - Set the colour for the different anchor points on the spline/arc. 默认值为（RGBA颜色） #FF0000FF。

  • Control point colour - Set the colour for each of the control point "handles" on an anchor point. 默认值为（RGBA颜色） #FF0000FF。

  • Tangent line colour - Set the colour for the lines linking the control points to the anchors. 默认值为（RGBA颜色） #00FFFFFF.

  • Selected point colour - Set the colour to be used for any selected anchor points on the spline/arc. 默认值为（RGBA颜色） #00FF00FF.

  • Point size - Set the size of the different anchor points and control points to be used. 默认值为10像素。
